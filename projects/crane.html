Things

While working at Crane, I was on the computer vision team, and we were tasked with training the AI to understand UI elements so it could code them for you. Part of this training was creating custom apps and feeding them to the machine. The apps shown here are a few selects from my time at Crane.

Fluga

Fluga is a concept for a flight finding app. The app is designed to show the quickest and cheapest flights first, getting the user where they need to go as quickly as possible.

Palette

Palette is an app that uses AI to identify emotions in user generated imagery. It then matches up the user imagery with corresponding film stills.

I designed Palette to be multi-platform. Rather than having to create an iOS and Android version, it was created to have a simple but universally recognizable design. To do this, I utilized Google's material principles, but I still changed them around enough to create palette's own aesthetic.

Spoon

Spoon is a mobile app concept that aggregates all the sports channels in one place. Users simply pay one fee and can watch every single sports game without a headache. No longer would one have to think, "Oh, do I have this channel?"

Font Fuck

I love exploring typography, so I wanted to make an app I would use. Often I find myself wondering what typeface someone is using when they post their work onto a site like behance or dribbble. Fontfuck is basically like those sites, however each post is titled and categorized by typeface.
