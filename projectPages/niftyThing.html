crane ai selects

while working at crane i was on the computer vision team. we were tasked with training the ai to understand ui elements so it could code them for you. part of this training was creating custom apps and feeding them to the machine. the apps shown here are a few selects from my time at crane.




fluga

Fluga is a concept for a flight finding app. The app is designed to show the quickest and cheapest flights first, getting the user where they need to go as quickly as possible.



palette

palette is an app that uses ai to identify emotions in user generated imagery. it then matches up the user imagery with corresponding film stills.

i designed Palette to be multi-platform. rather than having to create an ios and android version, it was created to have a simple but universally recognizable design. to do this, i utilized google's material guidelines while changing them around enough to create palette's own aesthetic.





spoon

spoon is an app concept that aggregates all the sports channels in one place. users simply pay one fee and can watch every single game without a headache. no longer would one have to think, "oh do i have this channel?"





font fuck

i love exploring typography, so i wanted to make an app i would use. often i find myself wondering what typeface someone is using when they post their work onto a site like behance or dribbble. fontfuck is basically like those sites, but each post is titled and categorized by typeface.
